Ultimate True American

Ultimate True American is a full stack Ruby on Rails application built as an aid for playing the game True American, popularized by Fox's New Girl. This mobile first app was built with Ruby on Rails and uses HTML5/CSS, Javascript/jQuery, Bootstrap, and oAuth for Facebook. Tess acted as the full-stack developer on this project, in charge of managing routing, authentication and used Easle.js to build a canvas drawing of the gameboard.

Outdoor Kitchen

Outdoor Kitchen is a place to find great gear for cooking and the outdoors. Built using Yeoman, MongoDB, ExpressJS, AngularJS, NodeJS and Socket.io. The API is built from scratch using MongoDB, Mongoose, local authentication and the Unsplash API. This piece is still in production, next steps are to add the Stripe API and finish the socket.io recipe sharing function.
